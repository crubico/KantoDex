# Kanto Pokédex

A web-based Pokédex for all 151 original Kanto Pokémon. Built mobile-first with live data from PokéAPI and animated GIF sprites.

## Live Site
[View the Pokédex](github.com/crubico/KantoDex)

## Features
- Search Pokémon by name or Pokédex number
- Filter by type
- Animated GIF sprites
- Click any card for full stats, height, weight, and abilities
- Featured carousel with 10 random Pokémon
- Animated hero gradient background

## Libraries Used
| Library | Purpose |
|---|---|
| [Anime.js](https://animejs.com/) | Card hover effects, modal open/close, grid animations |
| [Glide.js](https://glidejs.com/) | Touch/swipe carousel in the Featured section |
| [Granim.js](https://sarcadass.github.io/granim.js/) | Animated gradient canvas in the hero background |
| [Typed.js](https://mattboldt.com/demos/typed-js/) | Typewriter cycling text in the hero heading |

## Data Source
[PokéAPI](https://pokeapi.co) — free, open Pokémon data API
